						REPOSITORIO JAVA
_____________________________________________________________________________________________________________________
En este repositorio se incluyen distintas aplicaciones en Java, están creadosen proyectos para netbeans. 
_____________________________________________________________________________________________________________________
PROYECTOS INCLUIDOS:

-FicherosEnDirectorio. 	Aplicación que implementa un monitor java para la monitorización de un directorio 
 			y ejecuta una serie de acciones adicionales

-LectorXML. 		Aplicación que lee datos desde un fichero xml y los usa para la configuración del proyecto.

-OperacionesFicheros. 	Aplicación que hace operaciones básicas con ficheros.

-testExcel.	 	Aplicación para la lectura de ficheros excel y procesado de datos. 

-loaderMySQL.		Aplicación para la lectura desde una BD MySQL con JDBc en Java
